Stock data parsers:
  • {@link com.zigabyte.stock.parser.metastock.MetastockParser} parses historical stock data in Metastock directories.
  • {@link com.zigabyte.stock.parser.StockFilesSerializer} converts a metastock file tree to a serialized file for faster loading.
  • {@link com.zigabyte.stock.parser.SerializedStockFilesParser} loads data from a serialized file.
